About the App

This app is specially developed for teachers but anyone can use it. The StartClass application is to improve on the teaching and learning process which can also help to teach and learn both synchronously and asynchronously. It was selected to teach Noun, Pronoun, and Question Tags. This app defined all the above topics and explained the different types of each with examples. There is also an exercise at the end of each topic, which will give students an opportunity to practice what they have learnt. StartClass is a web based application which will teach students on Noun, Pronoun, and Question Tags.
